# Joconde Cultural Data Exploration

This project explores the French Joconde museum database, focusing on acquisition history,
artwork metadata, and institutional patterns.

## Objective
- Analyze museum acquisition data
- Clean and structure a complex cultural heritage dataset
- Build a lightweight data exploration app

## Data
- Source: Joconde database (French Ministry of Culture)
- High-dimensional metadata (artworks, authors, dates, locations, legal status)

## Methods
- Data cleaning and normalization
- Exploratory data analysis (EDA)
- Temporal and institutional analysis
- Interactive visualization with Streamlit

## Tools
Python, Pandas, Streamlit, Matplotlib

## Output
- Cleaned dataset
- Analysis notebook
- Streamlit application
